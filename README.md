**S2LSolutions** is a free, open source, flexible project management web application written using CakePHP.

#### Requirements
    * Apache with `mod_rewrite`
    	* Enable curl in php.ini
    	* Change the 'post_max_size' and `upload_max_filesize` to 200Mb in php.ini
    * PHP 5.3 or higher
    * MySQL 4.1 or higher
		* If STRICT mode is On, turn it Off.
